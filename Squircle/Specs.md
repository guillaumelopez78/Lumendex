# Squircle - Specs

## Design Tokens

All design tokens for this component are defined in the Lumen design system.

### Colors
- Primary

- Danger
- Neutral shades

### Typography
- Font family: Inter
- Font sizes (sm, md, lg)
- Font weight
- Line heights

### Spacing
- Padding (XS, SM, MD, LG)
- Margins
- Gap values
- Border width
