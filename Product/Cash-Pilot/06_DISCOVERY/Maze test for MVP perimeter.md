# Maze test for MVP perimeter

# Objectives

We aim to test the Cash Pilot MVP page to confirm that the page was understandable for users and no major alert could be raised by them.

The goal was to:

- Correct urgently the MVP if major issues were raised by users
- Write down minor issues raised by users in order to correct them after the MVP release

# Tools

A short Maze test was created an associated with a Figma prototype. It was a split between:

- Two challenges on the prototype
- Three questions to grade the challenges and the interface ergonomics
- One open question where the user could give his/her assessment

The Maze test: https://app.maze.co/projects/528298412/mazes/528298414/results?tab=results

The Figma prototype: https://www.figma.com/design/8o07VBIPRDjaXzOlZWoJQl/Cash-Pilot-MVP?node-id=1-105
