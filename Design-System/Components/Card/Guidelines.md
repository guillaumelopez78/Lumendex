## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Grouped related content | Single piece of content |
| Consistent padding + shadow | Varied spacing |
| Clickable: full card is link | Hidden link in text |
| Responsive: stack on mobile | Fixed width everywhere |
