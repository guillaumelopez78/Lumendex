## Tokens Used

| Property | Token | Value |
|----------|-------|-------|
| Padding | `--lumen-padding-16` | 16px |
| Border radius | `--lumen-border-radius-sm` | 4px |
| Shadow | `--lumen-shadow-sm` | 0 1px 3px |
| Background | `--lumen-color-surface-primary` | #FFFFFF |
| Border | `--lumen-color-border-neutral` | #E8E8E8 |

---

## Props

**Common:**
- `variant` — filled, outlined, text
- `color` — primary, accent, danger
- `size` — small, medium, large
- `disabled` — boolean
- `loading` — boolean
- `aria-label` — string

**Web:** className, style, onClick, onFocus, onBlur

**Mobile:** testID, onPress, accessible
