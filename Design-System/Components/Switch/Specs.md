## Tokens Used

| Property | Token | Value |
|----------|-------|-------|
| Height | `--lumen-sizing-44` | 44px |
| Width | `--lumen-sizing-64` | 64px |
| Border radius | `--lumen-border-radius-full` | 9999px |
| Transition | `--lumen-motion-standard` | 200ms |
| BG (on) | `--lumen-color-accent` | #0066FF |
| BG (off) | `--lumen-color-border-neutral` | #D0D0D0 |

---

## Props

**Common:**
- `variant` — filled, outlined, text
- `color` — primary, accent, danger
- `size` — small, medium, large
- `disabled` — boolean
- `loading` — boolean
- `aria-label` — string

**Web:** className, style, onClick, onFocus, onBlur

**Mobile:** testID, onPress, accessible
