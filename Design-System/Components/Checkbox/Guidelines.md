## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Multi-select saved via submit | Single instant toggle (use Switch) |
| Statement labels: "Accept terms" | Question labels: "Do you want...?" |
| Indeterminate for partial selection | No indeterminate state |
| Group with `<fieldset>` + `<legend>` | Checkboxes without group wrapper |
