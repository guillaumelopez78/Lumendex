## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Visible label always present | Placeholder-only label |
| Validate onBlur with error message | Real-time validation blocking entry |
| Label: noun, Placeholder: format | Label: verb, ALL CAPS |
| Show helper text when needed | Add caption to every field |
