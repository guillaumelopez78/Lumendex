## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Critical decisions only | Information alerts (use Alert) |
| Destructive action = danger button | Destructive + filled primary |
| Clear, focused action | Multiple unrelated actions |
| Scroll inside modal | Fixed modal height (cut content) |
