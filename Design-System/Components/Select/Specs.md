## Tokens Used

| Property | Token | Value |
|----------|-------|-------|
| Border radius | `--lumen-border-radius-sm` | 4px |
| Padding | `--lumen-padding-12` | 12px |
| Font size | `--lumen-font-size-body-md` | 14px |
| Height | `--lumen-sizing-40` | 40px |
| Border | `--lumen-color-border-neutral` | #D0D0D0 |
| Focus | `--lumen-color-border-focus` | #000000 |

---

## Props

**Common:**
- `variant` — filled, outlined, text
- `color` — primary, accent, danger
- `size` — small, medium, large
- `disabled` — boolean
- `loading` — boolean
- `aria-label` — string

**Web:** className, style, onClick, onFocus, onBlur

**Mobile:** testID, onPress, accessible
