## Do / Don't

| ✅ Do | ❌ Don't |
|---|---|
| Persistent, dismissible messages | Modal alerts (use Dialog) |
| Type: success, error, warning, info | Custom colors |
| Include action if needed | Multiple actions |
| Show error below form field | Alert far from context |
