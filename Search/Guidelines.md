# Search - Guidelines

## Guidelines

**Web vs Mobile**:
- Web: search icon on left, clear button on right, inline in toolbar
- Mobile: full-width search at top of page/section, same icon/clear pattern
- On mobile, keyboard is search-optimized (numeric/text as needed)
